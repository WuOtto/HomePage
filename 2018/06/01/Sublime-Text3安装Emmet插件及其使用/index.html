Sublime Text3安装Emmet插件及其使用

发表于 2018-06-01
介绍Sublime Text3安装Emmet插件及其使用。

安装

直接安装

直接下载安装包解压缩到Packages目录（菜单->preferences->Browse Packages）

使用Package Control组件安装

  1. 使用快捷键command + shift + p,调出Package Control;
  2. 输入install,点击Install Package;
  3. 使用快捷键command + shift + p,输入Emmet,点击安装;
  4. 等待安装完成。

使用

初始化文档

新建一个空白的html文件时,只需要输入!，然后按下Tab键就可以生成基础代码

轻松添加类、id、文本和属性

  1. 输入p.#foo,按下Tab自动补全

  2. 输入p.bar#foo,自动补全类和id

  3. 补全属性
    输入h1{foo},补全

  4. 输入a[href=#]，补全

嵌套

+:同级标签符号
