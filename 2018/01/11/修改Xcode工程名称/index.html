修改Xcode工程名称

发表于 2018-01-11

概述

有的时候需要在现有的项目上面开发一个新的项目，如果新建工程的话，就比较麻烦了，所以一般是直接现有的工程上面直接修改名字步骤如下：

1.修改工程名字

在这里修改完之后，会弹出一个对话框，点击Rename

2.修改Scheme名称

点击Xcode上面的工具栏Product->Secheme->Edit Secheme

直接回车修改名字，修改完成之后点击Close

3.修改目录名称

点击工程内需要修改名字的目录，回车直接修改

点击Find选择查找替换

将原来的工程名替换成你要修改的工程名

部分替换不了的，手动点进去修改

4.修改实体文件夹名称

在工程目录下，将对应的文件夹回车直接修改

5.修改.xcodeproj配置

找到你的项目.xcodeproj，右键显示包内容

双击打开.pbxproj文件

查找替换，将原来的工程名替换成你要修改的工程名

6.如果有pod

删除原来的pod文件三项

之后再重新pod install

结尾

这样就修改完成了
