Mac本地服务器Apache使用

在控制台开启服务

启动后，在浏览器中输入 http://127.0.0.1/ 或者 http://localhost 如果看到 It Works! 页面：

那么Apache 就启动成功了，站点的根目录为系统级根目录

启动后，你可以通过编辑 /etc/apache2/httpd.conf 文件来修改 Apache 配置。

访问服务器文件

1.将一个json文件（test.json）放到服务器路径下

2.用浏览器访问 http://127.0.0.1/test 或者
http://localhost/test

在浏览器中访问时，中文可能会出现乱码，但在项目中请求回来，打印在控制台的是正常的。
